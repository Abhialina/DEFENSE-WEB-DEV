HTML: Hyper Text Markup Language

  • HTML stands for HyperText Markup Language.

  • It is the standard language used to create and structure the content of a webpage.

What is an HTML Element?

  • :-An HTML element is defined by a start tag, some content, and an end tag:

  • Eg.- <tagname> Content goes here... </tagname>

HTML Attributes:

  1. The href Attribute:

    • The href attribute specifies the URL of the page the link goes to
    • Eg.- <a href="https://www.example.com" > Visit the site </a>

  2. The src Attribute:

    • The src attribute specifies the path to the image to be displayed
    • Eg.- <img src="image_1.png">

  3. The alt Attribute:

    • Display a default name when imgae can't load or image can't found.
    • It is used in basically <img> tag.
    • Eg. - <a src="image.png" alt="image_name" >

Let's Talk About first: Basic Tags of HTML

    1. Heading Tags(<h1> to <h6>):

    • <h1>Heading 1 </h1>

    • <h2>Heading 2 </h2>

    • <h3>Heading 3 </h3>

    • <h4>Heading 4 </h4>

    • <h5>Heading 5 </h5>
    • <h6>Heading 6 </h6>

    2. Paragraph Tag(<p>):

    Eg.-- <p> This is Paragraph Tag </p>

    3. Horizontal Rules Tag(<hr>):

    • The <hr> tag defines a thematic break in an HTML page, and is most often displayed as a horizontal rule.

    • The <hr> element is used to separate content in an HTML page:

    • Eg.-

      • This is the first paragraph

      This is other paragraph

    4. Line Break Tag(<br>):

    • The HTML <br> element defines a line break.

    • Purpose: Creates a single line break.
    • Eg.- Hello Sir,
      Kaise hai Aap.

    5. List Elements (<ul>, <ol>, <li>):

    • These tags are used to create lists.
    • <ul> : An Unordered List. It creates a bulleted list.
    • Eg.-
      • Tea Bag
      • Water
      • Sugar
      • Milk
    • <ol> : An Ordered List. It creates a numbered list.
    • Eg.-
      1. Coder Army
      2. Rohit Negi Sir
      3. Defence Batch
      4. HTML
    • <li> : A List Item. Each item in either type of list must be wrapped in an <li> tag.
    • Eg.- It is Nested tag using in <ul> or <ol> tag

    6. The Anchor Tag (<a>):

    • The Anchor tag is what makes the web "hypertext."
    • It is used to create a hyperlink to another webpage, a file, or a location within the same page.
    • <a href="https://www.coderarmy.in/" > Vist Coder Army </a>
    • Eg.- Vist Coder Army
